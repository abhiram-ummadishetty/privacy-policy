# Privacy Policy

**Effective Date:** 03/05/2026

## 1. Introduction

This application is a personal automation tool designed to assist with creating and publishing content to Pinterest.

## 2. Information We Collect

This application does not collect, store, or process any personal data from users.
All operations are performed locally on the user's machine.

## 3. Use of Information

The application uses locally stored images and generates captions using a local AI model.
No personal or sensitive data is transmitted to external servers, except for necessary communication with Pinterest APIs for publishing content.

## 4. Third-Party Services

This application interacts with Pinterest APIs to publish content.
Please refer to Pinterest’s Privacy Policy for more information on how they handle data.

## 5. Data Sharing

No user data is shared, sold, or distributed to third parties.

## 6. Security

Since all processing is done locally, data remains under the user’s control.
